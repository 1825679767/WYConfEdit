# WY配置编辑器

一款基于 Qt 的现代化配置文件编辑器，专为 AzerothCore World Server 配置文件设计。

## 功能特性

- **配置文件编辑** - 支持打开、编辑和保存 `.conf` 配置文件
- **中文翻译支持** - 通过 `translation.yaml` 提供配置项的中文名称和描述
- **分类筛选** - 左侧边栏按分类快速筛选配置项
- **搜索功能** - 支持按键名、名称、描述搜索配置项
- **悬停提示** - 鼠标悬停在数值列显示配置项描述
- **未保存提醒** - 关闭窗口时自动检测未保存的更改
- **记忆上次文件** - 自动记录并加载上次打开的配置文件
- **自适应分辨率** - 自动适配不同屏幕分辨率

## 环境要求

- Qt 5.x 或更高版本
- C++17 兼容的编译器

## 编译

```bash
qmake ConfEdit.pro
make
```

## 使用说明

### 打开配置文件
点击「打开配置」按钮，选择 `.conf` 配置文件

### 编辑配置项
双击表格中的任意行打开编辑对话框，可修改数值、中文名称、分类、描述

### 保存更改
点击「保存」按钮保存所有更改

### 搜索与筛选
- 在搜索框输入关键字搜索配置项
- 点击左侧分类列表筛选配置项

## 翻译文件格式

`translation.yaml` 文件格式：

```yaml
- key: RealmID
  section: 服务器
  name_zh: 服务器ID
  description_zh: |
    服务器的唯一标识符

- key: WorldServerPort
  section: 网络
  name_zh: 世界服务端口
  description_zh: 世界服务器监听的端口号
```
